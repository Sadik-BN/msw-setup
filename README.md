# msw-setup

MsO full setup free

Step-1: Uninstall old/deactivated mso from system
Step-2: Download ofiice setup :- <a href="https://c2rsetup.officeapps.live.com/c2r/download.aspx?ProductreleaseID=O365ProPlusRetail&platform=x64&language=en-us&version=O16GA"> Here</a>
Step-3: Open and run "OfficeSetup.exe" file. (It will download the mso)
Step-4: after finishing mso download run in powershell(admin) the command bellow:-
        # irm https://get.activated.win | iex
Step-5: Select Activation method # "Ohook - office"
Step-6: Select # "Install ohook office activation"
Step-7: Wait for activation. After successfull activation close cmd/powershel.
